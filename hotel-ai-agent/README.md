# Hotel AI Agent

An AI-powered n8n workflow designed to handle hotel-related automation tasks such as responding to customer queries, managing bookings, and providing hotel information.

## Features
- Handles hotel inquiries automatically
- AI-powered responses
- Can be connected to email, chat, or webhook triggers

## Tech Stack
- n8n
- OpenAI
- Webhooks / Email integrations

## Usage
1. Import the workflow JSON into n8n
2. Configure required credentials
3. Activate the workflow
